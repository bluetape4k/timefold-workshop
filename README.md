# timefold-workshop

[timefold.ai](https://timefold.ai/) 를 사용하는 예제를 제공하는 레포지토리입니다.

## 환경

- Java 21 이상
- Kotlin 2.1.0 이상
- Spring Boot 3.4.0 이상
- Kotlin Exposed 0.61.0 이상 (데이터 저장을 위해)
